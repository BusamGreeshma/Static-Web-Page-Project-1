# AETHERIA | Luxury Space Tourism & Orbital Voyages

A premium, immersive, and highly responsive static landing page built for **AETHERIA**, a futuristic space exploration and tourism agency. This portal guides prospective space travelers through booking luxury cabin tiers to orbits like Luna Sanctuary, Ares Prime on Mars, and Valkyrie Station in Jovian Orbit.

---

## 🌟 Key Features

- **Rich Futuristic Aesthetics**: Dark-mode design incorporating glowing neon primary accents (Teal & Purple HSL color spaces) and modern layouts.
- **Glassmorphic Interfaces**: Frosted panels built using CSS `backdrop-filter` rules for clean data cards and interactive forms.
- **Micro-Animations**: Slow-orbit rotations, dynamic card elevations, custom glowing input states, and transition hover states.
- **Responsive Layout**: Designed using CSS Flexbox and Grid, fully styled for smooth scaling on mobile, tablet, and widescreen viewports.
- **SEO-Optimized & Accessible**: Features complete metadata tags, correct document heading hierarchies, and inline SVG assets for speed and screen-reader accessibility.

---

## 🛠️ Technology Stack

- **Structure**: Semantic HTML5 markup
- **Styling**: Modern Vanilla CSS (using variables, flex/grid templates, and custom keyframes)
- **Typography**: Google Fonts (Space Grotesk & Inter)
- **Visuals**: Scalable Vector Graphics (SVG) and Custom High-Fidelity AI-generated imagery

## 💻 Local Setup & Development

To view and edit the project locally:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/BusamGreeshma/Static-Web-Page-Project-1.git
   cd Static-Web-Page-Project-1
   ```

2. **Open in Browser**:
   Double click the `index.html` file or launch it using your favorite live server extension (such as Live Server in VS Code).

3. **Check Code Quality**:
   Verify responsive behaviors by resizing the browser window or viewing through the browser's developer tools in responsive layout mode.
